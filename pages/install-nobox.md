# Install Nobox

Follow the instructions below to install Nobox in your app.

## Install Nobox-Client

Install nobox-client in your project: 

```shell
npm i nobox-client
```

## Get the API token
Go to [nobox.cloud](https://nobox.cloud), register and copy the token
